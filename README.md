# am100

Alpha Microsystems AM-100 Emulator 

Derived from the work of Mike Noel http://www.kicksfortso.com/Am100/

# Build
```
git clone https://github.com/8bitgeek/am100.git
cd am100
git submodule init
git submodule update
make
```
